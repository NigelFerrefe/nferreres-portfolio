<h1 align="center">Mi Portfolio Personal</h1>

---

## Descripción

Un portfolio profesional y versátil diseñado para mostrar mis habilidades, proyectos y experiencia. Cuenta con soporte para múltiples idiomas (Español e Inglés), un panel administrativo para gestionar contenido dinámico, modo oscuro/claro, y formulario de contacto.

##  Características

###  Multiidioma
- Soporte completamente funcional para **Español e Inglés**
- Cambio de idioma sin recarga de página
- Rutas dinámicas localizadas

###  Diseño Responsivo
- Interfaz moderna y atractiva
- Totalmente responsiva (Mobile-first)
- Tema oscuro/claro persistente


###  Secciones Principales
- **Home**: Presentación personal con CTA
- **Currículum**: Experiencia, educación, certificaciones y habilidades técnicas
- **Proyectos**: Galería de proyectos con detalles y enlaces
- **Contacto**: Formulario funcional con validación
- **Admin**: Panel privado para gestionar información dinámica
